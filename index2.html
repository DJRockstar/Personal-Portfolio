About

I didn't find technology. Technology found me!

I built my first website over 6 years ago. The site was super simple, but in hindsight, this was my technical singularity. More recently, I've realize that my quest into software engineering stems from the value that is added to one's life. Whether that value is added through building full stack applications using the latest and greatest tech stacks that breathe life into a user's computer screen or creating a microservice that fits coherently and seamlessly onto a product page, I feel proud of the user's experience.

I nerd out on creating React Components , RESTful APIs, Database Designs, Building Responsive Applications, Learning New Algorithms, and Solving Toy Problems.

When I am not coding, I am probably optimizing my health and body, hanging out with my nephews (4 yrs old and a newborn), reading biographies, dancing, and finding time to code on personal projects (yea, I know).
